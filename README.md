## Description

This repository contains the final project report for the Sentimazon project, developed as part of the Software Engineering course.

Sentimazon is a sentiment analysis project designed to analyze customer reviews and identify user opinions about mobile phone products. The project applies Natural Language Processing techniques to clean, process, analyze, and visualize review data.

The report describes the full development process of the project, including the project idea, requirements, design decisions, implementation, testing, CI/CD workflow, known issues, and possible future improvements.
